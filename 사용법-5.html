🏠 바탕화면(홈화면)에 설치하기
🤖 안드로이드 (삼성 갤럭시 등)
  1. 앱 주소를 크롬으로 열기
  2. 화면 맨 위 주소창 옆 또는
    우측 상단 점 세 개 ⋮ 누르기
  3. "홈 화면에 추가" 선택
  4. "추가" 버튼 누르기
  5. 홈 화면에 福 아이콘이 생겨요! ✅
🍎 아이폰 / 아이패드
  1. 앱 주소를 사파리로 열기
    (크롬 말고 반드시 사파리로!)
  2. 화면 하단 가운데 공유 버튼 □↑ 누르기
  3. 아래로 스크롤해서 "홈 화면에 추가" 선택
  4. 오른쪽 위 "추가" 누르기
  5. 홈 화면에 福 아이콘이 생겨요! ✅
📌 설치 후에는 인터넷 없이도 사용 가능해요!
앱처럼 전체화면으로 실행돼요.
🔊 중국어 소리 나오게 하기
🤖 안드로이드 (삼성 갤럭시)
  1. 설정 앱 열기
  2. 일반 관리 선택
  3. 언어 및 입력 선택
  4. 텍스트 음성 변환 선택
  5. Samsung TTS 또는 Google TTS 선택
  6. 언어 데이터 설치 누르기
  7. 목록에서 중국어(간체) 찾아서 다운로드
  8. 설치 완료 후 앱에서 🔊 버튼 누르면
    중국어로 읽어줘요! ✅
🍎 아이폰 / 아이패드
  1. 설정 앱 열기
  2. 손쉬운 사용 선택
  3. 말한 콘텐츠 선택
  4. 음성 선택
  5. 중국어(중국 본토) 선택
  6. 음성 데이터 다운로드
  7. 설치 완료 후 앱에서 🔊 버튼 누르면
    중국어로 읽어줘요! ✅
